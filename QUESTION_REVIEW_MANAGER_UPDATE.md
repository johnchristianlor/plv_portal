# Question Builder: Scroll and Review Manager Update

## Fixed

- Bulk Smart Paste now scrolls correctly inside the modal on desktop, tablets, and mobile devices.
- The Smart Paste header and action buttons remain visible while the content area scrolls.
- Long pasted questionnaires and long detected-question previews no longer lock the page.
- Touch scrolling and nested preview scrolling are supported.

## Added: Review Questions Manager

A **Review Questions** button is available in the Question Builder toolbar.

The review manager provides:

- One organized reading view for every saved question
- Questions grouped by section
- Search across question text, choices, answers, and section names
- Section filter
- Question-type filter
- Visible question count
- Clear display of points, choices, and answer keys
- Correct multiple-choice answer highlighting
- **Locate in builder** action that opens the correct section, scrolls to the question, and highlights it
- Responsive full-screen layout on small devices

No database migration is required.
